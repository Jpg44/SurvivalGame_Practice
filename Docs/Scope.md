# Scope (MVP)

## Pitch
- 3D 탑다운 시점 서바이벌 게임 루팅및 크래프팅으로 생존하여 탈출
- 탐색 -> 루팅 -> 전투 -> 탈출 아이템 획득 -> 탈출 
- 탈출 하기전까지 생존, 사망시 실패

## MVP Must-have 
01) 플레이어 이동, 탑다운 카메라
02) 플레이어 공격 방법 (근접무기, 원거리무기(투사체)슬롯전환)
03) 플레이어 스텟(체력, 스피드, 공격력 등)
04) 적 (스폰 -> 추적 -> 이동 -> 공격), 적 스탯 , 적 전투(플레이어&적 피격/죽음)
05) 아이템루팅 (주변 상자, 적 드랍)
06) 인벤토리, 드랍
07) UI(플레이어 스탯(HUD),적 체력바,인벤토리/크래프팅) / Scene(Title,Result) 이동 
08) 게임  저장(플레이어 위치, 아이템, 진행상황, 현재 스탯)
09) 크래프팅 기능(3~5)
10) 탈출 조건(정보아이템 획득 -> 헬기 생성 -> 열쇠 획득 -> 탈출)(탈출 아이템 위치고정)

- 미니맵
- 낮 밤
- 맵 크기 강화
- 

## Non-goals (이번엔 안 함)
- 멀티 구현
- 건축 기능(크래프팅 강화)
- 무기 다양화
- 맵 강화(절차 생성맵)
- 런처

## Definition of Done
- MVP 기능체크
- 빌드 및 실행 방법 README + 60초 소개영상
- 6 ~ 10 분 플레이 (시작 -> 인게임 -> 탈출 성공or실패)
- 버그및 크래시 ZERO
